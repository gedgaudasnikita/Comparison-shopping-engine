Detailed Description

Implemented by the classes that aim to encapsulate the behaviour of certain server endpoints. Provides an additional layer in handling the request, used for IO data preparation and handling. When implementing this interface, it is highly recommended to inherit from some IEndpoint<RequestBody, RequestParameters, ResponseBody> implementing class and delegate as much work as possible to the methods implemented by it.
